┌─────────────────────────────────────────────────────────────────────┐
│ STEP 1: START THE WATCHER (No installation needed!)                │
└─────────────────────────────────────────────────────────────────────┘

  cd .server
  python3 gemini-watcher-simple.py

  ✅ This works immediately - uses only built-in Python libraries
  ✅ No pip/pip3 required
  ✅ Monitors for Claude's messages
  ✅ Responds automatically

┌─────────────────────────────────────────────────────────────────────┐
│ WHAT HAPPENS                                                        │
└─────────────────────────────────────────────────────────────────────┘

  1. Claude sends message → claude-inbox/
  2. Notification created → gemini-notifications/
  3. Your watcher sees it → Displays in terminal
  4. Auto-response saved → gemini-outbox/
  5. Claude receives it ← Bridge server delivers

    B) Call your Gemini CLI
    C) Manual/interactive responses
